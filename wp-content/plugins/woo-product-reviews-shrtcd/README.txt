== Installation ==

This section describes how to install the plugin and get it working.

1. Download and activate the plugin through the "Plugins" menu in WordPress
2. Click on "How to use" link below plugin title, yo will be redirected to Plugin's Help Page
3. Copy any shortcode from examples
4. Paste shortcode on any page or post content
5. Make changes for your needs
